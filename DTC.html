Auditory Feedback Project




Design Project



About This Project

Our client was a visually impaired, but wanted to run the Chicago Marathon. He struggled to use treadmills because he could not read the display or use the button to adjust the speed. We created a system that provided audial feedback so that our client could easily train for the marathon on a treadmill.
